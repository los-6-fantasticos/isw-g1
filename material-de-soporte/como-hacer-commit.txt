git checkout master (partimos de master como base)
git pull origin master (nos aseguramos tener master actualizado)
(si no hay conflictos)
git checkout -b isw-nn-xxx (creamos y nos movemos al nuevo branch)
(se hacen todos los cambios)
git status (vemos los cambios)
git add . (marcamos todos los cambios para comitear)
git commit -m "adds a lot of changes" (generamos commit con mensaje descriptivo)
git log (vemos nuestro commit)
git push origin isw-nn-xxx (enviamos nuestro branch al repositorio remoto)
